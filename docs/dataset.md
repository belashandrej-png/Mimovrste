## 2.4 Обоснование полноты выборки

### Временной охват:
- Данные охватывают период **с ноября 2021 по декабрь 2023 года** (26 месяцев)
- Это позволяет:
  - Проанализировать влияние инфляции 2022 года на розничные цены
  - Выявить сезонные паттерны (предновогодний рост, летние распродажи)
  - Построить долгосрочные тренды по категориям

### Объём и репрезентативность:
- **480+ миллионов записей** обеспечивают статистическую значимость результатов
- Представлены **все основные категории** товаров маркетплейса (электроника, одежда, товары для дома и др.)
- Достаточно данных для выявления **редких паттернов** и аномалий
