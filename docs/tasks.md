# 2. Задачи анализа

## 2.1. Статистические задачи (Business Intelligence)

### Задача 1: Анализ динамики цен по категориям (2021-2023)
**Тип:** Статистическая
**Метод:** Группировка по годам и категориям, расчет средних цен
**Результат:** График динамики для топ-10 категорий

### Задача 2: Анализ сезонности цен
**Тип:** Статистическая
**Метод:** Группировка по месяцам, расчет индексов сезонности
**Результат:** Выявление пиковых и минимальных периодов

### Задача 3: Расчет индекса потребительских цен (ИПЦ)
**Тип:** Статистическая
**Метод:** Индексный анализ с базой январь 2021 = 100
**Результат:** График ИПЦ с динамикой роста цен

### Задача 4: Анализ волатильности цен по категориям
**Тип:** Статистическая
**Метод:** Расчет коэффициента вариации (CV)
**Результат:** Ранжирование категорий по стабильности цен

### Задача 5: Анализ распределения цен
**Тип:** Статистическая
**Метод:** Гистограммы, расчет квартилей
**Результат:** Статистика распределения по топ-5 категориям

### Задача 6: Анализ структуры ассортимента
**Тип:** Статистическая
**Метод:** Расчет долей категорий в общем ассортименте
**Результат:** Круговая диаграмма топ-10 категорий

### Задача 7: Анализ цен по брендам
**Тип:** Статистическая
**Метод:** Группировка по брендам, расчет медианных цен
**Результат:** Ранжирование брендов по ценовому сегменту

## 2.2. Исследовательские задачи (Data Science)

### Задача 8: Выявление аномальных скачков цен в 2022 году
**Тип:** Исследовательская
**Метод:** Метод стандартных отклонений (2σ)
**Результат:** Обнаружение месяцев с аномальными изменениями

### Задача 9: Выявление выбросов в ценах
**Тип:** Исследовательская
**Метод:** Метод межквартильного размаха (IQR, метод Тьюки)
**Результат:** Выявление некорректных данных (11.81% выбросов)

### Задача 10: Индексный анализ по методу Ласпейреса
**Тип:** Исследовательская
**Метод:** Индекс Ласпейреса с учетом структуры ассортимента
**Результат:** Корректированный индекс цен

### Задача 11: Ранжирование категорий по инфляции
**Тип:** Исследовательская
**Метод:** Расчет темпов роста по категориям
**Результат:** Топ категорий с наибольшей инфляцией

### Задача 12: Определение оптимальных периодов покупок
**Тип:** Исследовательская
**Метод:** Анализ сезонности с расчетом потенциальной экономии
**Результат:** Рекомендации по времени покупок для каждой категории
