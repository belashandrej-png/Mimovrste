# 6. Визуализация результатов

## 6.1. Требования к визуализации

Согласно методическим указаниям (стр. 37), результаты исследований должны быть представлены в виде набора графиков и диаграмм с соблюдением следующих требований:

| Требование | Реализация в проекте |
|------------|---------------------|
| Подписанные оси с размерностями | Все графики имеют подписи осей с указанием единиц измерения (евро, год, месяц) |
| Числовые характеристики входных данных | Для каждого графика указано количество обработанных строк (50 002) |
| Выводы под каждым графиком | К каждому графику добавлены текстовые выводы и интерпретация |
| Читаемость подписей | Шрифт 10-12 pt, разрешение 300 DPI для печати |
| Нумерация слайдов | Все слайды презентации пронумерованы |

## 6.4. Пример оформления графика с выводом

**График:** Динамика средних цен по топ-10 категориям (2021-2023)

![price_trend_categories](../reports/figures/price_trend_categories.png)

**Числовые характеристики:**
- Объём выборки: 50 002 строки
- Период: ноябрь 2021 — декабрь 2023
- Категории: топ-10 по количеству записей

**Выводы:**
1. Наиболее высокий рост цен продемонстрировала категория "erotika" (+111.6%), что может быть связано с изменением структуры спроса в период 2022 года.
2. Категория "svetila luci" показала снижение цен на 55.3%, что указывает на возможную ценовую конкуренцию или изменение ассортимента.
3. Общий тренд: рост цен в 7 из 10 категорий, что коррелирует с макроэкономической ситуацией (инфляция в ЕС).

## 6.5. Интерактивная визуализация (опционально)

Для расширения функционала предусмотрена возможность экспорта данных в формат, совместимый с инструментами бизнес-аналитики:

- **Qlik Sense / QlikView**: CSV-файлы с агрегированными данными
- **Tableau**: Extract-файлы (.hyper)
- **Power BI**: PBIX-шаблон с подключением к Parquet

## 6.6. Соответствие требованиям методички
