# 7. Программная реализация

## 7.1. Организация программного решения

Программная реализация курсового проекта выполнена в виде модульного Python-приложения с использованием системы контроля версий **Git** и хостингом на платформе **GitHub**.

### Ссылка на репозиторий:

https://github.com/belashandrej-png/mimovrste-analysis


### Языки и технологии:

| Компонент | Технология | Обоснование |
|-----------|------------|-------------|
| Язык программирования | Python 3.9+ | Богатая экосистема библиотек для анализа данных, простота прототипирования |
| Обработка данных | Pandas, NumPy | Эффективная работа с табличными данными, векторизованные операции |
| Визуализация | Matplotlib, Seaborn | Гибкость настройки, экспорт в высокое разрешение (300 DPI) |
| Хранение промежуточных данных | Apache Parquet (zstd) | Высокая степень сжатия, быстрый доступ к колонкам |
| Система контроля версий | Git + GitHub | Отслеживание изменений, возможность совместной работы, воспроизводимость |
