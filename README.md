Описание проекта:
Анализ динамики цен и структуры ассортимента товаров интернет-магазина Mimovrste.
Проект направлен на исследование изменения цен по категориям товаров, выявление сезонных колебаний и анализ волатильности цен.
Цель проекта:
Комплексный исследовательский анализ данных о ценах и ассортименте товаров с использованием статистических методов.
В рамках проекта проводится анализ динамики цен, сезонности, волатильности цен, ранжирование категорий по уровню инфляции и определение оптимальных периодов покупок.
Источник данных:
Данные получены из открытых источников и используются для анализа ценовых изменений и структуры ассортимента товаров. Официальный источник: https://www.kaggle.com/datasets/thunderpede/testststs/data

## Правила работы с репозиторием

### 1. Создание Pull Request
Перед созданием Pull Request необходимо синхронизировать ветку с актуальной версией `main`.

Каждый Pull Request должен соответствовать одной логической задаче и содержать описание внесённых изменений.

### 2. Название Pull Request и веток
Название Pull Request и веток должно:

- быть на английском языке
- быть написано в нижнем регистре
- слова должны разделяться символом `_`

Примеры названий:

- `add_price_volatility_analysis`
- `update_readme_structure`
- `add_category_inflation_ranking`

### 3. Работа с ветками
Разработка ведётся в отдельных ветках.

Перед началом работы необходимо создать новую ветку от актуальной версии `main`.

После завершения задачи создаётся Pull Request в ветку `main`.

# Описание проекта

Проект посвящён анализу потребительских цен и ассортимента товаров интернет-магазина **Mimovrste** на основе данных маркетплейса.

Исследование направлено на выявление динамики цен, сезонных колебаний и особенностей изменения стоимости товаров по категориям.

---

# Цель проекта

Провести исследовательский анализ данных для выявления закономерностей изменения цен и структуры ассортимента.

В рамках проекта выполняются:

- анализ цен на товары
- исследование сезонности и волатильности цен
- анализ ассортимента товаров
- описательная статистика
- индексный анализ
- ранжирование категорий товаров по уровню инфляции
- определение оптимальных периодов покупок
- выявление категорий с аномальной волатильностью

---

# Основные этапы анализа

1. Сбор данных и анализ цен Mimovrste  
2. Анализ волатильности и сезонности цен  
3. Анализ ассортимента Mimovrste  
4. Эмпирический анализ  
   - описательная статистика

   - индексный анализ

5. Ранжирование категорий по инфляции  
6. Определение оптимальных периодов покупок по категориям

# Запуск проекта

1. Клонировать репозиторий
git clone https://github.com/belashandrej-png/dataOzon.git

2. Перейти в папку проекта
cd dataOzon

3. Создать виртуальное окружение
python -m venv venv

4. Активировать окружение
Windows:
venv\Scripts\activate

Mac / Linux:

source venv/bin/activate

5. Установить зависимости
pip install -r requirements.txt

6. Запустить Jupyter Notebook
jupyter notebook

После этого можно открыть ноутбук с анализом данных.
