ЧТО ЭТО:
Полностью автоматический проект аналитики.
При запуске сам скачает данные и откроет дашборд.

ТРЕБОВАНИЯ:
✓ Windows 10/11
✓ Python 3.8+ (https://python.org/downloads/)
✓ Интернет (для скачивания данных)
✓ 2 ГБ свободного места

КАК ЗАПУСТИТЬ:

1. Распакуйте ZIP-архив в любую папку

2. Установите Python (если нет):
   - Скачайте с python.org/downloads/
   - При установке ПОСТАВЬТЕ галочку "Add Python to PATH"

3. Дважды кликните на файл: START.bat

4. Подождите 5-10 минут:
   - Установятся библиотеки
   - Скачаются данные из облака
   - Откроется браузер с графиками

5. Готово! Работайте с дашбордом

ЕСЛИ НЕ РАБОТАЕТ:

❌ Ошибка "Python не найден":
   → Установите Python с python.org
   → Поставьте галочку "Add Python to PATH"
   → Перезагрузите компьютер

❌ Данные не скачиваются:
   → Откройте вручную: https://cloud.mail.ru/public/RM8x/JTvZztfUR
   → Скачайте файл mimodump-dataset.csv
   → Положите его в папку: data/

❌ Ошибка при запуске:
   → Запустите START.bat от имени администратора
   → Правой кнопкой → "Запуск от имени администратора"

ДАШБОРД ОТКРОЕТСЯ ЗДЕСЬ:
http://localhost:8501

Для остановки:
- Закройте браузер
- В черном окне нажмите Ctrl+C
