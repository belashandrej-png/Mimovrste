# Набор данных

## Источник данных

**Наименование:** Mimovrste 2021-2023 Item Prices

**URL:** https://www.kaggle.com/datasets/govzegunns/mimovrste-2021-2023-item-prices

**Лицензия:** CC0 1.0 (Public Domain)

## Характеристики данных

| Параметр | Значение |
|----------|----------|
| Объем сырых данных | 26 ГБ |
| Количество записей | ~480 900 277 |
| Количество колонок | 21 |
| Период данных | Ноябрь 2021 - 2023 |
| Формат | CSV (разделитель ';') |
| Кодировка | UTF-8 |

## Структура данных

### Основные поля:

1. **name** (string) - название товара
2. **price** (float) - текущая цена в евро
3. **current_price** (float) - актуальная цена
4. **lowest_price** (float) - минимальная цена
5. **msrp_price** (float) - рекомендованная производителем цена
6. **image** (string) - URL изображения
7. **item_url** (string) - URL товара
8. **category_name** (string) - категория товара
9. **category_url** (string) - URL категории
10. **available** (boolean) - доступен ли товар
11. **in_promotion** (boolean) - есть ли акция
12. **lowest_product_price** (float) - минимальная цена товара
13. **on_sale** (boolean) - распродажа
14. **free_delivery** (boolean) - бесплатная доставка
15. **product_id** (string) - ID продукта
16. **review_count** (int) - количество отзывов
17. **review_stars** (float) - рейтинг в звездах
18. **short_desc** (string) - краткое описание
19. **brand_name** (string) - бренд
20. **item_url_key** (string) - ключ URL
21. **parsed_at** (datetime) - дата и время парсинга

## Обоснование полноты выборки

1. **Временной охват:** Данные охватывают период с ноября 2021 по 2023 год, что позволяет:
   - Проанализировать влияние инфляции 2022 года
   - Выявить сезонные паттерны
   - Построить долгосрочные тренды

2. **Объем данных:** 
   - 480+ миллионов записей обеспечивают статистическую значимость
   - Представлены все основные категории товаров маркетплейса
   - Достаточно данных для выявления редких паттернов

3. **Репрезентативность:**
   - Mimovrste - один из крупнейших маркетплейсов Словении
   - Широкий ассортимент (электроника, одежда, товары для дома и др.)
   - Данные актуальны для исследования розничных цен в ЕС

## Обработка данных

### Предварительная очистка:
- Удаление дубликатов
- Преобразование цен в числовой формат
- Удаление некорректных значений (отрицательные цены, выбросы)
- Стандартизация дат

### Выборка для анализа:
- Размер выборки: 50 002 записи (0.01% от общего объема)
- Метод выборки: случайная стратифицированная выборка
- Сохранение пропорций по категориям
